# AI Blog Chat Generator
## based on codedex project tutorial

A simple Python project that generates blog-style text using the OpenAI API.

## Files

- `chat_generator.py`: main Python script
- `.env`: stores the API key locally (not uploaded)
- `.gitignore`: ignores sensitive and unnecessary files

## Setup

1. Install dependencies
2. Create a `.env` file
3. Add your API key:
   API_KEY=your_api_key_here
4. Run the Python file
